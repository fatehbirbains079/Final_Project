# CS30 Final Project Proposal

# Project Title
Math Escape Game

# Purpose of the Program
The purpose of this program is to make a simple text based game where the player must escape a building by answering math questions.

# Story Line
The player is trapped inside a locked building. Each door in the building is locked and can only be opened by solving a math question. The player has three chances to answer questions correctly. If the player unable to solve the problem in three attempts then the player would be stuck in the building.

# Theme
Puzzle / Escape

# Example Catch Phrases
- "Solve the question to unlock the door."
- "Correct! The door opens."
- "Wrong answer. You lost a chance."
- "No chances left. You are stuck."
- "You escaped the building!"

# Target User
The target user is all the student or anyone who likes simple puzzle games. The game is easy to understand and does not require advanced math skills.

# Must-Have Features
- A player with 3 chances
- Math questions using basic operations
- A menu system for starting or quitting the game
- Win and lose conditions
- Player information 

# Nice-To-Have Features
- Different difficulty levels
